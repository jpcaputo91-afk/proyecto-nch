# Recursos recomendados

## Sitios web

- **TreasuryDirect** (treasurydirect.gov) — Sitio oficial del Tesoro de EE.UU. para comprar bonos
- **FRED** (fred.stlouisfed.org) — Base de datos económica de la Fed de St. Louis
- **Investopedia** (investopedia.com) — Diccionario y guías financieras

## Libros

- *The Bond Book* — Annette Thau (introducción completa a bonos)
- *Fixed Income Securities* — Bruce Tuckman (más técnico)
