# Proyecto NCH — Aprendizaje de Mercados Financieros

Repositorio personal para aprender sobre el mercado bursátil, con foco en **activos de renta fija de Estados Unidos**.

---

## Objetivo

Construir una base sólida de conocimiento sobre cómo funcionan los mercados financieros, entendiendo en profundidad los instrumentos de renta fija del mercado estadounidense: qué son, cómo se compran, cómo se valúan, y cómo se usan como herramienta de inversión.

---

## Estructura del repositorio

| Carpeta | Contenido |

| `01-conceptos/` | Fundamentos del mercado bursátil y renta fija |
| `02-instrumentos/` | Tipos de instrumentos: Treasuries, bonos, etc. |
